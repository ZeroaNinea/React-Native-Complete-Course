# React Native Complete Course

This is a learning project that contains a complete guide about React Native. It covers everything from the basics to advanced topics, including navigation, animations, and more.

## Getting Started

To get started with this project, you should install dependencies and run the project with Expo.

```bash
npm i
npm start
```
